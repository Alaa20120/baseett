# نظام إدارة الفواتير المتقدم (Advanced Arabic Invoice & Business Management System)

نظام ويب متكامل لإدارة العمليات التجارية والمالية، صُمم خصيصاً لتلبية احتياجات الشركات والمناديب في بيئة عمل تدعم اللغة العربية بشكل كامل (RTL) مع ربط مباشر وقوي مع ملفات Excel لإدارة البيانات.

## 🚀 نظرة عامة على المشروع والهيكلية (Project Overview & Architecture)

### وصف المشروع
يوفر النظام واجهة مستخدم حديثة وتفاعلية لإدارة المبيعات، المشتريات، المخزون، والعهود المالية. يتميز النظام بالسرعة والأمان والقدرة على العمل دون الحاجة لخادم backend معقد، حيث يعتمد على تخزين البيانات محلياً وتصديرها لتقارير إكسيل احترافية.

### قائمة المزايا والوحدات (Feature List)
يحتوي النظام على 14 وحدة متكاملة:
1.  **لوحة التحكم (Dashboard):** عرض إحصائيات فورية ورسوم بيانية للأداء.
2.  **الخزينة (Treasury):** إدارة التدفقات النقدية (الوارد والصادر).
3.  **المبيعات (Sales):** تسجيل فواتير البيع مع دعم المرتجع والتحصيل الجزئي.
4.  **المشتريات (Purchases):** إدارة عمليات الشراء من الموردين وتأثيرها على المخزن.
5.  **المخزون (Inventory):** مراقبة رصيد المنتجات في المخزن الرئيسي ومستودعات المناديب.
6.  **العملاء (Customers):** قاعدة بيانات شاملة للعملاء مع تتبع المديونيات.
7.  **الموردين (Suppliers):** إدارة حسابات الموردين وفواتير الشراء.
8.  **المناديب (Reps):** نظام متقدم لتتبع أداء المناديب وعهدهم المالية ومخازنهم.
9.  **الموارد البشرية (HR):** إدارة الموظفين، الرواتب، والسلف.
10. **كشوفات الحساب (Statements):** إنشاء كشوفات حساب مفصلة لكل الجهات.
11. **التقارير (Reports):** تصدير تقارير إكسيل شاملة وتحليلات الربحية.
12. **المنتجات (Products):** تعريف المنتجات، وحداتها، وأسعارها.
13. **تفاصيل الفواتير (Invoice Detail):** عرض تفصيلي لكل حركة مالية.
14. **ملفات التعريف (Entity Profile):** صفحات متخصصة لكل عميل أو مورد أو مندوب.

### الحزمة التقنية (Technology Stack)
-   **JavaScript:** لغة البرمجة الأساسية باستخدام نمط IIFE Modules.
-   **Chart.js v4.4.1:** للمخططات والرسوم البيانية التفاعلية.
-   **SheetJS (XLSX) v0.20.1:** لقراءة ومعالجة ملفات Excel.
-   **ExcelJS v4.3.0:** لإنشاء ملفات Excel احترافية مع تنسيق الجداول.
-   **QRCodeJS:** لتوليد رموز QR Code للفواتير (متوافق مع متطلبات الزكاة والدخل).
-   **Font Awesome v6.5.1:** للأيقونات والواجهة البصرية.
-   **Fonts:** Cairo (للنصوص العربية)، Inter (للأرقام والتنسيقات اللاتينية).

### هيكلية النظام (Architecture)
يعتمد النظام على بنية ثنائية الطبقات (Two-Tier):
1.  **طبقة UI Controller (`app.js`):** المسؤولة عن التفاعل مع المستخدم، تحديث الواجهة، وإدارة الـ Modals.
2.  **طبقة البيانات (`excel-engine.js`):** المحرك الأساسي (ExcelEngine) الذي يدير كافة العمليات الحسابية، منطق العمل، وحفظ البيانات.

### حفظ البيانات والصلاحيات
-   **الاستمرارية:** يتم حفظ البيانات تلقائياً في `localStorage` للمتصفح مع إمكانية استيراد وتصدير ملفات Excel كنسخ احتياطية كاملة.
-   **نظام الصلاحيات (RBAC):**
    -   **مشرف (Admin):** وصول كامل لكافة التقارير، الخزينة، وتعديلات المخزن.
    -   **مندوب (Rep):** وصول محدود لعمليات المبيعات والتحصيل الخاصة به فقط مع عرض مخزنه الشخصي.

## 📊 هيكل نموذج البيانات (Data Model Structure)

### مجموعات البيانات (Collections)
يحتوي كائن `workbookData` على 17 مجموعة بيانات أساسية:
- `sales`, `purchases`, `inventory`, `customers`, `reps`, `repPayments`, `products`, `suppliers`, `staff`, `hr`, `customerPayments`, `treasury`, `vouchers`, `expenses`, `repWarehouse`, `warehouseTransfers`, `categories`.

### العلاقات الأساسية (Relationships)
- **المبيعات ← العملاء ← دفعات العملاء:** كل عملية بيع تزيد المديونية، وكل دفعة تقللها.
- **المشتريات ← الموردين ← المخزون:** الشراء يزيد مديونية المورد ويزيد رصيد المنتج.

### الآثار الجانبية التلقائية (Side-Effects)
1.  **الخزينة:** تسجيل تلقائي لأي حركة نقدية.
2.  **المخزن:** خصم تلقائي عند البيع (من مخزن المندوب أو الرئيسي).
3.  **الحسابات:** تحديث فوري لأرصدة العملاء والموردين.

## 🛠 مرجع البرمجة (Public API Reference)

يتم تصدير المهام في `ExcelEngine` عبر عدة مجالات:
- **الأمن:** `login`, `getCurrentUser`, `logout`
- **العمليات:** `addSale`, `addPurchase`, `addCustomer`, `addProduct`
- **المخزن:** `transferToRepWarehouse`, `returnFromRepWarehouse`, `getRepWarehouse`
- **الخزينة:** `logTreasuryTransaction`, `getTreasuryBalance`, `addVoucher`

### الأمن والتوثيق
- **التوقيع الرقمي:** استخدام `generateInvoiceSecurity` لإنشاء `XML` و `Hash` لكل فاتورة لضمان عدم التلاعب.

## 📱 التثبيت على الهاتف (PWA)

يمكنك الآن تثبيت البرنامج على هاتفك الأندرويد أو الآيفون واستخدامه كتطبيق مستقل:
1.  افتح رابط الموقع من المتصفح (Chrome للاندرويد، Safari للايفون).
2.  اضغط على **"إضافة إلى الشاشة الرئيسية" (Add to Home Screen)**.
3.  سيظهر التطبيق كأيقونة مستقلة على شاشتك.
4.  يمكنك استخدام التطبيق حتى بدون اتصال بالإنترنت بفضل تقنية **Service Worker**.
